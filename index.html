¿Qué es Arduino?

Arduino es una plataforma de hardware y software abierta para crear proyectos electrónicos. Aquí encontrarás una introducción, guías para instalar el entorno, ejemplos de código y recursos para aprender.

Instalación del IDE de Arduino

  1. Visita https://www.arduino.cc y descarga el IDE (o usa el Arduino Web Editor).
  2. Instala los drivers si tu placa lo requiere (Windows suele pedirlos la primera vez).
  3. Conecta tu placa Arduino por USB, abre el IDE y selecciona la placa y el puerto en el menú Herramientas.
  4. Prueba subiendo el ejemplo "Blink" desde Archivo → Ejemplos → 01.Basics → Blink.

Cómo programar: conceptos básicos

Explicaciones claras y ejemplos prácticos para comenzar con Arduino sin abrumarte.

1. Estructura básica

setup() → se ejecuta una vez; loop() → se repite. Evita bloquear el loop para mantener la respuesta del sistema.

2. Entradas y salidas

  • pinMode(pin, MODE): INPUT / INPUT_PULLUP / OUTPUT.
  • digitalRead / digitalWrite: señales digitales (HIGH/LOW).
  • analogRead(pin): lectura analógica (0..1023).
  • analogWrite(pin, value): PWM (0..255) en pines compatibles.

3. Temporización: delay vs millis

delay() detiene todo; millis() permite temporizar sin bloquear y controlar varias tareas.

4. Ejemplos prácticos

Blink (simple)

// void setup() { pinMode(LED_BUILTIN, OUTPUT); }
// void loop() { digitalWrite(LED_BUILTIN, HIGH); delay(1000); digitalWrite(LED_BUILTIN, LOW); delay(1000); }

Blink no bloqueante (millis)

// const int led = LED_BUILTIN;
// unsigned long prev = 0;
// unsigned long interval = 500;
// void setup(){ pinMode(led, OUTPUT); }
// void loop(){
//   unsigned long now = millis();
//   if(now - prev >= interval){
//     prev = now;
//     digitalWrite(led, !digitalRead(led)); // alterna LED
//   }
//   // aquí puedes leer sensores o atender botones sin bloquear
// }

5. Lectura de botones y debounce

Usa INPUT_PULLUP para simplificar el cableado y aplica un debounce (10–50 ms) en software para lecturas estables.

Resumen rápido

  • Inicia Serial.begin(9600) para depurar.
  • Usa constantes descriptivas: const int LED = 13;
  • Evita delay() si necesitas multitarea.

Checklist antes de subir

  • Placa y puerto seleccionados en el IDE.
  • Cable USB y alimentación OK.
  • Polaridad del LED y conexiones verificadas.
