¿Qué significa "if" en Arduino?

La instrucción if permite tomar decisiones en el programa: si una condición es verdadera ejecuta un bloque; si no, puede usar else o else if para otras condiciones. En Arduino se usa para reaccionar a lecturas (sensores, botones) y controlar salidas (LEDs, relés) según criterios.

Ejemplo de uso típico: leer un valor analógico y encender un LED si supera un umbral.

Paso a paso

  1. Decide la entrada (botón, potenciómetro, sensor) y la salida (LED).
  2. Conecta el sensor y configura pines en setup().
  3. En loop() lee la entrada y aplica una condición: if(valor > umbral){ digitalWrite(led,HIGH);} else {digitalWrite(led,LOW);}.
  4. Ajusta el umbral y prueba con el simulador para comprobar la respuesta.
  5. Agrega else if para múltiples rangos/acciones si es necesario.
