Proyecto 11: LED RGB (Cátodo Común)

Inicio Proyectos

Materiales

  • Placa Arduino (UNO, Nano, etc.)
  • LED RGB cátodo común
  • 3 resistencias 220Ω (una por canal)
  • Cables y protoboard
  • Cable USB

¿Qué es un LED RGB de cátodo común?

Un LED RGB cátodo común tiene el cátodo (masa) compartido y tres ánodos separados para rojo, verde y azul. Conectando el cátodo a GND y cada ánodo a un pin PWM del Arduino (mediante resistencias) puedes variar la intensidad de cada color con analogWrite() y mezclar colores.

Ejemplo básico: usar pines PWM 9(R),10(G),11(B) y modificar valores 0–255 para obtener distintas mezclas.

// const int r = 9, g = 10, b = 11;
// void setup(){ pinMode(r, OUTPUT); pinMode(g, OUTPUT); pinMode(b, OUTPUT); }
// void loop(){
//   analogWrite(r, 255); analogWrite(g, 0); analogWrite(b, 0); // rojo
//   delay(800);
//   analogWrite(r, 0); analogWrite(g, 255); analogWrite(b, 0); // verde
//   delay(800);
//   analogWrite(r, 0); analogWrite(g, 0); analogWrite(b, 255); // azul
//   delay(800);
// }

Paso a paso

  1. Conecta el cátodo común a GND.
  2. Conecta los tres ánodos (R,G,B) a pines PWM mediante resistencias 220Ω.
  3. En setup() configura los pines como OUTPUT.
  4. Usa analogWrite(r,g,b) para variar cada canal 0–255 y mezclar colores.
  5. Prueba paletas básicas (rojo, verde, azul, amarillo, blanco) en el simulador.
