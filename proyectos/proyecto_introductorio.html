Proyecto introductorio - Encender un LED

Inicio Proyectos

Introducción

En este proyecto introductorio aprenderas a encender y pagar un led mediantea un simulador integrado.

Ejemplo: lectura de botón con debounce

// const int BTN = 2;
const int LED = 13;
unsigned long lastChange = 0;
const unsigned long debounce = 40;
int state = HIGH;
void setup(){ pinMode(BTN, INPUT_PULLUP); pinMode(LED, OUTPUT); }
void loop(){
  int reading = digitalRead(BTN);
  if(reading != state && (millis() - lastChange) > debounce){
    state = reading;
    lastChange = millis();
    digitalWrite(LED, state == LOW ? HIGH : LOW);
  }
}

Resumen rápido

  • Inicia Serial.begin(9600) para depurar.
  • Usa constantes descriptivas: const int LED = 13;
  • Evita delay() si necesitas multitarea.
