¿Qué es el Monitor Serial?

El Monitor Serial permite comunicar la placa Arduino con el ordenador por el puerto serie. En el código usamos Serial.begin(baud) en setup() para iniciar la comunicación (ej. 9600). Luego enviamos texto con Serial.print() o Serial.println(). Podemos leer datos con Serial.available() y Serial.read() para recibir comandos desde el ordenador.

Usos comunes: depuración (imprimir valores de sensores), comunicación con el usuario (entradas vía teclado) y ver paso a paso el comportamiento del programa.

Práctica en simulador

Abre el simulador, carga el ejemplo y observa cómo el Monitor Serial muestra mensajes. Puedes modificar mensajes o enviar caracteres desde la interfaz de Wokwi.

Solución (simulador)

Paso a paso

  1. Conecta la placa por USB y abre el Monitor Serial en el IDE.
  2. En setup() llama Serial.begin(9600);.
  3. Usa Serial.print() o Serial.println() para enviar mensajes o valores desde el loop().
  4. Para leer datos desde el ordenador usa if(Serial.available()){ char c = Serial.read(); }.
  5. Prueba enviar comandos desde el monitor y haz que el sketch reaccione (encender un LED, cambiar un parámetro).
