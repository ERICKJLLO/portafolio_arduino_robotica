¿Qué es un potenciómetro?

Un potenciómetro es una resistencia variable con tres patas: dos extremos y un cursor (wiper). Al girarlo cambia la resistencia entre el cursor y cada extremo, por lo que el voltaje en el cursor varía. Conectar un extremo a 5V, el otro a GND y el cursor a una entrada analógica permite leer valores 0–1023 con analogRead().

Ejemplo de código (leer potenciómetro y controlar brillo)

Leer el valor analógico y mapearlo a 0–255 para usar analogWrite() en un pin PWM:

// const int potPin = A0;
// const int ledPin = 9; // pin PWM
// void setup(){ pinMode(ledPin, OUTPUT); }
// void loop(){
//   int v = analogRead(potPin);           // 0..1023
//   int out = map(v, 0, 1023, 0, 255);    // 0..255
//   analogWrite(ledPin, out);
//   delay(20);
// }

Paso a paso

  1. Conecta el potenciómetro: extremos a 5V y GND, cursor (wiper) a A0.
  2. Conecta un LED con resistencia a un pin PWM (ej. 9) y GND.
  3. En loop() lee analogRead(A0), mapea 0..1023 a 0..255 con map().
  4. Pasa el valor mapeado a analogWrite(ledPin, out); y prueba girando el potenciómetro.
  5. Ajusta filtros (promedio) si deseas un cambio de brillo más suave.
