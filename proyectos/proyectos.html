Proyecto 1 - Encender un LED

Aprende a encender y apagar un LED con Arduino (digitalWrite, pinMode).

Abrir

Proyecto 2 - Contador binario

Contador binario con 4 LEDs (muestra valores 0–15 usando bits).

Abrir

Proyecto 3 - Control de brillo (analogWrite / PWM)

Controla el brillo de un LED usando analogWrite() (PWM). Aprende a mapear valores 0–255 para ajustar el duty cycle y ver el efecto en el LED.

Abrir

Proyecto 4 - Monitor Serial

Introducción al Monitor Serial: enviar y recibir datos usando Serial.begin(), Serial.print() y Serial.println().

Abrir

Proyecto 5 - Potenciómetro (Resistencia variable)

Leer un potenciómetro con analogRead() y usar el valor para controlar el brillo de un LED con analogWrite().

Abrir

Proyecto 6 - Uso de IF en Arduino

Aprende a usar if, else if y else para tomar decisiones; ejemplo: leer un potenciómetro o sensor y encender/apagar un LED según un umbral.

Abrir

Proyecto 7 - Potenciómetro: Controlar un LED

Usa un potenciómetro para variar el brillo de un LED; aprenderás a leer analogRead(), mapear valores y aplicar analogWrite().

Abrir

Proyecto 8 - Ciclo FOR en Arduino: Secuencia de LEDs

Aprende a usar el ciclo for para recorrer pines y crear secuencias (efecto "carrera" o "chaser") con varios LEDs.
