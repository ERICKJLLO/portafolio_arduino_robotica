Proyecto 13: Conectar un zumbador activo

Inicio Proyectos

Materiales

  • Placa Arduino (UNO, Nano, etc.)
  • Zumbador activo (buzzer)
  • Resistencia 220Ω (opcional según buzzer)
  • Cables y protoboard
  • Cable USB

Descripción

Un zumbador activo incorpora un oscilador interno: basta con aplicarle HIGH o LOW desde un pin digital para que suene. Para tonos más controlados se usan zumbadores pasivos con tone(), pero aquí mostramos cómo conectar y activar un zumbador activo desde un pin digital simple.

// const int buzzer = 8;
// void setup(){ pinMode(buzzer, OUTPUT); }
// void loop(){
//   digitalWrite(buzzer, HIGH); // suena
//   delay(300);
//   digitalWrite(buzzer, LOW);  // silencio
//   delay(300);
// }

Solución (simulador)

Paso a paso

  1. Conecta el zumbador activo al pin digital (ej. 8) y GND (resistencia si recomienda el fabricante).
  2. En setup() configura el pin como OUTPUT.
  3. Activa con digitalWrite(buzzer, HIGH) y apaga con LOW. Para tonos use tone(pin, frecuencia) si es pasivo.
  4. Prueba ritmos simples (encendido/apagado con delay) y luego tonos con tone() si aplica.
  5. Comprueba volumen y duración antes de usar en clase.
