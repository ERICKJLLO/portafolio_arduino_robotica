¿Qué hace el ciclo for?

El ciclo for repite un bloque de código muchas veces cambiando una variable de control. Es útil para recorrer arrays o pines y crear secuencias. Por ejemplo, con varios LEDs puedes encender uno tras otro usando un for para indexar los pines.

Ejemplo de código (secuencia simple):

// const int leds[] = {8,9,10,11,12,13};

//   for(int i=0;i<6;i++){

//     delay(120);
//     digitalWrite(leds[i], LOW);
//   }

Solución (simulador)

Paso a paso

  1. Conecta varios LEDs (4–6) a pines sucesivos con resistencias a GND.
  2. En setup() configura los pines como OUTPUT.
  3. En loop() usa un for para recorrer el array de pines: encender, delay(), apagar.
  4. Prueba invertir el sentido o cambiar tiempos para diferentes efectos.
  5. Experimenta con for + millis() para versiones no bloqueantes.
