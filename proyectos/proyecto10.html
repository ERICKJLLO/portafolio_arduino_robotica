Proyecto 10: Leer valores desde el Monitor Serial

Inicio Proyectos

Materiales

  • Cable USB
  • LED (opcional para pruebas)
  • Resistencia 220Ω (si usa LED)
  • Cables y protoboard

¿Qué significa leer desde el Monitor Serial?

El Monitor Serial permite enviar caracteres desde el ordenador hacia la placa. En el código se usa Serial.available() para comprobar si hay datos y Serial.read() o Serial.readString() para leerlos. Con esto puedes recibir comandos, números o texto y reaccionar (ej.: encender/apagar un LED o ajustar parámetros).

Ejemplo simple: si llega la letra '1' enciende un LED, si llega '0' lo apaga.

// const int led = 13;
// void setup(){ pinMode(led, OUTPUT); Serial.begin(9600); }
// void loop(){
//   if(Serial.available()){
//     char c = Serial.read();
//     if(c == '1') digitalWrite(led, HIGH);
//     else if(c == '0') digitalWrite(led, LOW);
//   }
//}

Paso a paso

  1. Conecta la placa y abre el Monitor Serial.
  2. En setup() inicia Serial.begin(9600);.
  3. Comprueba con if(Serial.available()) y lee con Serial.read() o Serial.readString().
  4. Procesa el dato recibido y ejecuta acciones (ej.: '1' enciende LED, '0' lo apaga).
  5. Prueba desde la consola del simulador o el IDE y observa la respuesta.
