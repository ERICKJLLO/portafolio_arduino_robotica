Proyecto 1: Encender un LED

Inicio Proyectos

Objetivo

Aprender a encender un LED con Arduino usando digitalWrite y pinMode.

Materiales

  • Placa Arduino (UNO, Nano, etc.)
  • LED
  • Resistencia 220Ω
  • Cables y protoboard

Montaje

Conectar el ánodo del LED (pierna larga) al pin 13 (o el pin que elijas) mediante la resistencia; el cátodo a GND.

Código de ejemplo

// void setup() { pinMode(13, OUTPUT); }
// void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }
