¿Qué hace el ciclo while?

El ciclo while ejecuta repetidamente un bloque de código mientras la condición sea verdadera. Es útil cuando no se sabe de antemano cuántas veces se repetirá la acción (ej.: esperar a que un sensor alcance un valor). Evita bucles infinitos usando condiciones que eventualmente cambien.

Ejemplo simple: encender un LED mientras un botón esté presionado.

// const int btn = 2;
// const int led = 13;
// void setup(){ pinMode(btn, INPUT_PULLUP); pinMode(led, OUTPUT); }
// void loop(){
//   while(digitalRead(btn) == LOW){ // mientras botón presionado
//     digitalWrite(led, HIGH);
//   }
//   digitalWrite(led, LOW);
//}

Solución (simulador)

Paso a paso

  1. Conecta un botón o sensor y un LED con resistencia a un pin.
  2. En loop() usa while(condición){ ... } para repetir acciones mientras la condición sea verdadera.
  3. Ejemplo: mientras el botón esté presionado, enciende el LED; al soltarlo, apágalo.
  4. Cuidado con bucles infinitos: asegúrate que la condición pueda cambiar.
  5. Prueba en el simulador y añade mensajes en Serial para depurar.
