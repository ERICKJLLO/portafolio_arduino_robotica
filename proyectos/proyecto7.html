Proyecto 7: Usar un potenciómetro para controlar un LED

Inicio Proyectos

Materiales

  • Placa Arduino (UNO, Nano, etc.)
  • Potenciómetro (10k recomendado)
  • LED
  • Resistencia 220Ω
  • Cables y protoboard
  • Cable USB

Descripción

Conecta el potenciómetro entre 5V y GND y el cursor al pin analógico (por ejemplo A0). Lee el valor con analogRead() (0–1023), conviértelo a 0–255 con map() y pásalo a analogWrite() en un pin PWM para controlar el brillo del LED.

Video explicativo

Ejemplo de código

// const int potPin = A0;
// const int ledPin = 9; // PWM pin
// void setup(){ pinMode(ledPin, OUTPUT); }
// void loop(){
//   int val = analogRead(potPin);         // 0..1023
//   int out = map(val, 0, 1023, 0, 255);  // 0..255
//   analogWrite(ledPin, out);             // controla brillo
//   delay(20);
// }

Paso a paso

  1. Conecta potenciómetro (5V/GND/wiper a A0) y LED en pin PWM con resistencia.
  2. Configura pinMode(ledPin, OUTPUT) en setup().
  3. Lee valor con analogRead(A0), mapea y aplica analogWrite(ledPin, out);.
  4. Prueba y calibra el rango si el brillo no se ajusta linealmente.
